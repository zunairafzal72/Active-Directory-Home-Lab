# Creating User Accounts in Active Directory

Creating user accounts is one of the most common tasks performed by IT support teams.

In this lab environment, users were created through the Active Directory Users and Computers console.

## Steps

1. Open Active Directory Users and Computers
2. Expand the domain company.local
3. Right-click the Users container
4. Select New
5. Click User

## User Information

Example user:

First Name: John  
Last Name: Smith  
User logon name: jsmith

A password must then be assigned.

Common options include:

User must change password at next logon  
Password never expires  
Account is disabled

For this lab, the "User must change password at next logon" option was selected to simulate a typical corporate setup.

## Result

The user account was successfully created and can now authenticate to computers joined to the domain.
