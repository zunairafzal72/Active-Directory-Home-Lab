# Resetting a User Password

Password resets are one of the most frequent requests handled by help desk staff.

This lab demonstrates how a password reset can be performed using Active Directory Users and Computers.

## Steps

1. Open Active Directory Users and Computers
2. Navigate to the Users container
3. Locate the user account
4. Right-click the account
5. Select Reset Password

Enter the new password and confirm it.

It is also possible to force the user to change their password during their next login.

## Result

The password reset was successful and the user can log into the domain using the updated credentials.
